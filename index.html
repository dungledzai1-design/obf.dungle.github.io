API Tích Hợp

Hướng Dẫn Sử Dụng API

Đăng nhập và vào mục API Key để lấy key của bạn

# Lấy danh sách dịch vụ
GET /api/v2?action=services&key=YOUR_API_KEY

# Kiểm tra số dư
GET /api/v2?action=balance&key=YOUR_API_KEY

# Tạo đơn hàng
POST /api/v2?action=add&key=YOUR_API_KEY&service=1&link=URL&quantity=1000

# Kiểm tra trạng thái đơn
GET /api/v2?action=status&key=YOUR_API_KEY&order=ORDER_ID

# Response mẫu (balance):
{"balance":"1500000","currency":"VND"}

# Response mẫu (add order):
{"order":"123456"}
